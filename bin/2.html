文件上云说明：

1、流程：client<-->oss<-->callback

2、优点：大幅减轻appServer上行及下行的访问及流量压力，支持缩略图

3、缺点：不支持多文件上传，目前暂时不支持水印

4、获取的oss临时访问权限时间为1小时，超时后app需要重新获取

获取附件上传方式接口[编辑]

服务地址: {authUrl}/upload/getRequestParameter.do

请求参数[编辑]

Key 必须 意义
fileName 是 文件名称

请求URL样例[编辑]

{authUrl}/upload/getRequestParameter.do?userStr={"fileName":"test_201608.jpg"}


说明： 

     通用参数请参考 接口通用说明

返回参数[编辑]

Key 类型 意义
data JSON对象 返回具体数据
desc String 描述，如 normal
lastUpdateTime String 本次数据的生成时间
result String 成功标识 0：成功 -1：失败

JSON数据字段说明[编辑]

Key 数据类型 意义
uploadType String 上传类型：0代表上传到oss，1代表上传到本地
bucketName String oss的bucketName
endPoint String oss的上传文件地址
imgEndPoint String oss图片访问基地址
objectKey String 文件key，oss唯一标识
accessKeyId String oss临时访问身份的id
accessKeySecret String oss临时访问身份的secret
securityToken String oss临时访问身份的授权信息
callbackUrl String oss上传文件的回调地址，需要在前面拼上服务器地址
callbackHost String oss回调的host
callbackHost String 超时时间

oss上传文件说明[编辑]

1、oss直传文件请参看：https://help.aliyun.com/document_detail/32055.html?spm=5176.doc32008.6.345.tVhoay

2、setCallbackBody中需要拼接如下信息：

Key 数据类型 意义
entCode String 企业编码
uid String 用户id
fileName String 文件名
fileSize String 文件大小
filePath String 文件路径，oss图片服务基地址 + "/"＋objectKey

3、addCallbackVar中要加入上面描述的回调参数变量，格式callback.addCallbackVar("x:entCode", "entCode");
